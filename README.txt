# 5G Backhaul Analysis
This repository contains code and analysis for evaluating 5G backhaul performance using hybrid RF/FSO systems. The analysis includes:
- Exploratory Data Analysis (EDA)
- Data balancing methods
- Predictive modeling using specific and generic models

## Requirements
- Python 3.7+
- Libraries: pandas, matplotlib, scikit-learn

## Running the Code
1. Clone the repository: `git clone https://github.com/<your-username>/<repository-name>.git`
2. Navigate to the folder: `cd <repository-name>`
3. Run the code: `python analysis.py`
